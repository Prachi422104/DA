# Task 5 - Exploratory Data Analysis (EDA)

## Objective
To analyze Titanic dataset and extract insights using statistical and visual exploration.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Females had higher survival rate.
- First class passengers survived more.
- Fare positively correlated with survival.
- Age had slight impact on survival.

## Conclusion
EDA helped in identifying patterns, trends and important features affecting survival.
